# 🎓 Lernify - E-Learning Platform

Lernify is a full-stack e-learning web application built to provide a simple, interactive, and user-friendly learning experience. It allows users to explore courses, register/login securely, and access learning content in an organized way.

---

## 🚀 Live Demo  

Frontend: https://e-learning-platform-lernify.vercel.app
Backend: https://e-learning-platform-lernify-backend.onrender.com

---

## 📌 Features

- User registration and login system  
- Secure authentication using JWT 
- Password encryption for security  
- Course listing and detailed course pages  
- Responsive design for all screen sizes  
- RESTful APIs using Node.js and Express  
- MongoDB database integration  
- Simple and clean UI for better user experience

## ⚙️ Installation & Setup

### Clone repository
git clone https://github.com/maihvish-Siddiqui/E-Learning-Platform-Lernify.git

---

### Backend setup
cd backend  
npm install  

Create .env file:

PORT=5000  
MONGO_URI=your_mongodb_connection_string  
JWT_SECRET=your_secret_key  

Run backend:
npm start  

---

### Frontend setup
cd frontend  
npm install  
npm start  

---

## 🔗 API Endpoints

POST /api/auth/register → Register user  
POST /api/auth/login → Login user  
GET /api/courses → Get all courses
